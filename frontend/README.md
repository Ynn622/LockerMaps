### 啟動前端 (Frontend)

```bash
cd frontend

# 安裝相依套件
npm install

# 啟動開發用伺服器
npm run dev

# 啟動開發用伺服器 (區域訪問)
npm run dev -- --host
```
啟動後，請瀏覽 `http://localhost:5173`。
